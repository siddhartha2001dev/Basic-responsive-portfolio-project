  • Flap Quest (Responsive flappy bird clone):


    Flap Quest is an interactive 2D browser-based game inspired by the classic Flappy Bird. Developed entirely using HTML, CSS, and vanilla JavaScript, this project showcases efficient use of DOM manipulation, event handling, and real-time animations to deliver a smooth and engaging gameplay experience.The objective of the game is simple yet challenging: guide the bird through an endless series of pipes without colliding. The game dynamically adjusts difficulty over time, encouraging player focus and reflexes. It features responsive design principles to ensure compatibility across various screen sizes and modern web browsers. Also I designed the assets used in this project by myself with the help of procreate for 2D designs graphics images.
